   - If not, update it to use `var(--blue-color)`

   - Change the `background-color` to a lighter shade of green
   - Use `#66BB6A` as the new hover color
   - The line should look like this:
     ```css
     background-color: #66BB6A;
     ```

6. **Update link color**
   - Find the `a` selector in the CSS file
   - Verify it uses `var(--blue-color)` for the color property
   - If not, update it to use `var(--blue-color)`

7. **Review and test**
   - Save the `styles.css` file
   - Open `index.html` in a web browser
   - Verify that all previously blue elements are now green
   - Check that the sidebar, buttons, and links have changed color
   - Ensure the button hover effect shows a lighter shade of green

8. **Fine-tune if necessary**
   - If any elements don't look right with the new green color, adjust their specific styles as needed
   - Pay special attention to contrast and readability

9. **Commit changes**
